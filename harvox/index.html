I am a full stack web developer with particular proficiency in back end development. I make particular use of the MEVN stack. The MEVN stack consists of MongoDB (and Mongoose), ExpressJS, VueJS and NodeJS.

MongoDB is the most popular NoSQL database. Data is stored in documents rather than in a data table. This makes it particularly great for storing data that may not be homogeneous.

ExpressJS is a NodeJS library that is used for backend web development. It helps with serviceslike routing, APIs, etc.

VueJS is a frontend framework for building responsive web pages using components. It is a scaled-down version of Angular and an alternative to ReactJS.

NodeJS is a JavaScript runtime environment that uses a C engine to run JavaScript outside a a web browser. It is the 2nd most popular option for backend development.
